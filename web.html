My Projects

A few of the projects that I have completed.

Platform Jumper

This is a small functional Platform Jumper that implements the PyGame library. The game takes inspiration from two classic games, Mario and Gravity Guy. This game has simple mechanics but requires creative thinking in order to successfully finish all levels. The game has a basic UI for the all key event screens.

Object Detection App

This application recognizes and labels a variety of objects but its main objective is to identify and locate cars. This was a project that was used to help Toyota streamline their vehicle production by allowing AI to manage their processes with more precision. The app uses a model which has been trained using a comprehensive sample pool.

Sorting Conveyor Belt

This program sorts different packages into different categories and visually represents this process implementing the Java AWT library. This project was created in response to a hypothetical business scenario in which a shipping company needed assistance in identifiying and sorting packages to increase efficiency.
